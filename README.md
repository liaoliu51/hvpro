# hv-analysis-pro

> 横纵分析法 Pro - 廖势无双个人增强版

## 📖 简介

横纵分析法（Horizontal-Vertical Analysis Pro）是由**廖势无双**提出的深度研究方法论，融合了：
- 语言学中的历时 - 共时分析（Saussure）
- 社会科学中的纵向 - 横截面研究设计
- 商学院案例研究法
- 竞争战略分析的核心思想

在原版基础上新增**4 个分析维度**，形成更完整的上市公司研究框架。

## ✨ Pro 版新增功能（v2.0）

| 维度 | 分析内容 | 数据来源 |
|------|----------|----------|
| 🔍 最赚钱业务分析 | 分业务收入 + 毛利率 + 收入占比变化 | 东方财富妙想 |
| 🔄 转型情况分析 | 战略转型路径 + 动因 + 成效 + 风险 | 公告 + 财报 |
| ⚖️ 优势劣势对比 | SWOT 框架（技术/产品/市场/成本/管理/财务） | 竞品对比 |
| 📈 股价弹性对比 | 涨幅/波动率/估值/资金偏好/催化剂敏感度 | 行情数据 |

## 🎯 适用场景

- ✅ 上市公司深度研究（A 股/港股/美股）
- ✅ 产品/公司/概念/人物系统性分析
- ✅ 竞品对比与行业格局研究
- ✅ 投资决策前的深度尽调

## 📊 分析框架

```
纵轴（时间深度）          横轴（同期广度）
    ↓                        ↓
起源 → 发展 → 转型       竞品 A vs 竞品 B vs 竞品 C
    ↓                        ↓
    └────→ 交汇洞察 ←─────┘
           ↓
    未来推演（三剧本）
```

## 🚀 使用方法

### 1. 安装依赖

```bash
pip install weasyprint markdown --break-system-packages
```

### 2. 配置环境变量（仅上市公司分析需要）

```bash
export MX_APIKEY="mkt_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

### 3. 运行分析

```bash
# 通过 OpenClaw 调用
# 触发词：横纵分析、研究一下、帮我分析、深度研究等

# 或直接调用 PDF 生成脚本
python scripts/md_to_pdf.py input.md output.pdf --title "报告标题" --author "廖势无双"
```

## 📄 报告结构

1. **一句话定义** - 用一句话说清楚研究对象
2. **纵向分析** - 从诞生到当下的完整历程（6000-15000 字）
3. **横向分析** - 竞争图谱与竞品对比（3000-10000 字）
4. **横纵交汇洞察** - 交叉分析与未来推演（1500-3000 字）
5. **金融数据分析** - 六个维度深度分析（2000-5000 字，仅上市公司）
6. **信息来源** - 所有引用来源清单

## 📈 篇幅要求

| 部分 | 字数范围 |
|------|----------|
| 纵向分析 | 6,000 - 15,000 字 |
| 横向分析 | 3,000 - 10,000 字 |
| 横纵交汇 | 1,500 - 3,000 字 |
| 金融数据分析 | 2,000 - 5,000 字 |
| **全文总计** | **12,000 - 35,000 字** |
